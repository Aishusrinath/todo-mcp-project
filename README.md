# Simple To-Do App

This is a small proof-of-concept to-do app built for the MCPs and Skills Across the Software Lifecycle assignment.

## Features
- Add tasks
- Display tasks in a list
- Prevent empty task submission
- Clear input after adding a task

## Required Skill Workflow
- grill-me
- write-a-prd
- prd-to-issues
- tdd
- improve-codebase-architecture

## MCP Demo
Playwright was used as part of the MCP demonstration.

## Run
Open `index.html` in a browser.
